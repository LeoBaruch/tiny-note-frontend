# Tiny Note 演示说明

## 🚀 快速开始

### 1. 启动应用

```bash
npm run dev
```

应用将在 `http://localhost:3000/tiny-note` 启动
应用的路由都在 `/tiny-note` 下


### 2. 演示账户

为了演示功能，我们提供了以下演示账户：

**登录信息：**

- 邮箱：`demo@example.com`
- 密码：`demo123`

**或者注册新账户：**

- 访问 `/tiny-note/register` 页面
- 填写用户名、邮箱和密码
- 系统会自动创建新账户并登录

## ✨ 功能演示

### 用户认证

1. **登录页面** (`/tiny-note/login`)

   - 使用演示账户登录
   - 体验表单验证
   - 查看错误处理

2. **注册页面** (`/register`)
   - 创建新账户
   - 密码确认验证
   - 自动登录

### 笔记管理

1. **笔记列表** (`/tiny-note/`)

   - 查看预设的示例笔记
   - 体验搜索功能
   - 按标签和分类筛选
   - 分页导航

2. **创建笔记** (`/tiny-note/note/new`)

   - 使用富文本编辑器
   - 设置标题、分类、标签
   - 公开/私密设置

3. **编辑笔记**

   - 点击笔记卡片上的编辑按钮
   - 修改内容和属性
   - 实时保存

4. **删除笔记**
   - 点击删除按钮
   - 确认对话框
   - 从列表中移除

### 富文本编辑器

1. **文本格式化**

   - 粗体、斜体、下划线
   - 代码块
   - 快捷键支持

2. **块级元素**

   - 标题（H1、H2、H3）
   - 列表（有序、无序）
   - 引用块
   - 文本对齐

3. **快捷键**
   - `Ctrl+B`：粗体
   - `Ctrl+I`：斜体
   - `Ctrl+U`：下划线
   - `Ctrl+\``：代码

### 标签和分类

1. **标签管理**

   - 预设标签：工作、学习、生活、技术
   - 不同颜色区分
   - 笔记关联

2. **分类管理**
   - 预设分类：工作笔记、学习笔记、生活记录、技术文档
   - 笔记组织
   - 快速筛选

### 搜索功能

1. **实时搜索**

   - 标题和内容搜索
   - 搜索结果高亮
   - 支持中文

2. **高级筛选**
   - 按标签筛选
   - 按分类筛选
   - 组合筛选

### 响应式设计

1. **桌面端**

   - 完整功能展示
   - 侧边栏可折叠
   - 宽屏优化

2. **移动端**
   - 触摸友好
   - 垂直布局
   - 自适应组件

## 🚧 注意事项

### 演示限制

1. **数据持久化**：当前使用内存存储，刷新页面数据会重置
2. **后端 API**：使用模拟数据，实际部署需要真实后端
3. **文件上传**：图片上传功能暂未实现
4. **用户管理**：用户数据仅存储在本地

### 性能说明

1. **加载速度**：模拟网络延迟，实际环境会更快
2. **内存使用**：编辑器组件占用一定内存
3. **响应性**：大文档编辑时可能有轻微延迟
